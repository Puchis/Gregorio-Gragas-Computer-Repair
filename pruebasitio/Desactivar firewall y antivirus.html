Desactivar firewall y antivirus

Desactivar firewall

Puede que el problema de no poder conectarte radique en el software del firewall que estés usando. Una forma sencilla de comprobar esto es desactivándolo e intentar entrar en Internet. Si te deja, ya sabemos dónde está el fallo. Si sigues igual, entonces es que no era del Firewall.

Los pasos para desactivar un firewall dependen del que utilices, pero si tienes problemas para desactivarlo, lo más rápido es desactivar todo el software de firewall que se ejecuta en el equipo de esta manera:

  1. En el cuadro de búsqueda de la barra de tareas, escribe Símbolo del sistema, haz click en Símbolo del sistema y, después, selecciona Ejecutar como administrador > Sí.
  2. En el símbolo del sistema, escribe netsh advfirewall set allprofiles state off y, después dale a Entrar.
  3. Si seguimos sin conexión de Red y queremos activar de nuevo todos los firewalls que estén instalados, en el símbolo del sistema debemos poner netsh advfirewall set allprofiles state on y darle a enter.

Desactivar antivirus

Al igual que con el método anterior, prueba a desactivar antivirus y software anti-malware para comprobar si puedes o no conectarte a Internet. Y si no estás seguro exactamente de qué programa de software antivirus o de prevención de malware tienes instalado, es posible que Windows pueda decírtelo de esta manera:

  1. En el cuadro de búsqueda de la barra de tareas, escribe Sistema y seguridad.
  2. Selecciona Revisar el estado del equipo y resolver los problemas.
  3. Selecciona la flecha abajo situada junto a Seguridad. Si Windows puede detectar el software antivirus, este se mostrará en Protección antivirus. Para el software de prevención de malware, busca en Protección contra spyware y software no deseado.
